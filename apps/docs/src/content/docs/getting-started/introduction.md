# Minions Conversations

Email threads, reply parsing, intent detection, and escalation triggers

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-conversations/sdk`) — Core library for Node.js
- **Python SDK** (`minions-conversations`) — Python bindings
- **CLI Tool** (`@minions-conversations/cli`) — Command-line interface
- **Web App** — Interactive playground at [conversations.minions.wtf](https://conversations.minions.wtf)
