---
title: Installation
description: Install Minions Conversations packages
---

## TypeScript / Node.js

```bash
npm install @minions-conversations/sdk minions-sdk
```

## Python

```bash
pip install minions-conversations
```

## CLI

```bash
npm install -g @minions-conversations/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
